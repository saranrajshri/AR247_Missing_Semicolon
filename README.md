## Installation

Clone the repository.

```
git clone https://github.com/saranrajshri/SIH-2020.git
```

## Install the dependencies
Install the node_modules for both client and server.
```
yarn install

cd client
yarn install


```

## Usage
```
yarn run dev

cd client
yarn start
```
